MtoolsDataEvents

  1. Home
  2. Modules
  3. MtoolsDataEvents
Description

This is the Merjent Fulcrum Tools Module to use within a Fulcrum Application's data event scripting environment.

The Mtools singleton provides all methods from the following classes through inheritance:

  • Utils - Core utility methods
  • FulcrumHelpers - Fulcrum form and field helpers
  • RequestHelpers - HTTP request helpers
  • DataEventHelpers - Data event specific methods (weather, fire danger, etc.)
