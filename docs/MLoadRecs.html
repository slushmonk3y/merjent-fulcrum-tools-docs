new MLoadRecs()

Loads records from a Fulcrum form

Description

Loads the form schema and creates dictionary to convert form_value keys to data_name keys. This will be modified soon to utilize the new query filtering options in LOADRECORDS.

Properties
Name Type Description
updateCallback function

Function to call to process retrieved records
recIDs Array

Array of string _record_ids to use in LOADRECORDS.
appFvs Object

Dictionary converting form_value keys to data_names
appRecords Array

Array of loaded records with form_value keys converted to data_name keys.
