Report Mtools

  • Use FIELDNAMES({data_name}) to keep report functional with changes in data_names
  • Use M.arrRem or M.mkMflds({arr},{fields to exclude}) to remove fields not wanted visible
  • Generation Scripts designed to work by section, so fields outside of section are not printed

Default Grid Notes

  • Default grid is 12 cells wide.
  • Use option '-hn' to have fields hidden if nul
MfldColl.m('data_name').opt = '-hn'
  • Use MfldColl to generate Default Grid
  • Use MfldColl.m to adjust options and span
MfldColl.m('data_name').span = 2

fldVal() Method

Yes/No Fields

Yes/No fields are automatically converted to checkboxes by default

Choice Fields

Choice fields by default display as a concatenation of strings separated by ", ". To convert display of choice fields to checkboxes add option '-cb' or '-hcb' to Mfld.

MfldColl.m('data_name').opt = '-cb'
