Utility helper methods

Description
For use in any JavaScript project, including Google Apps scripts etc.
Method naming uses the following conventions:
arr = Array
dt = Date
iso = String of a date in javascript ISO 8601 format (YYYY-MM-DD or YYYY-MM-DD**T**hh:mm:ss**Z**)
str = String
rows = Array of Objects(or JSONs)
dn = String of a Fulcrum *data_name* of a table field
