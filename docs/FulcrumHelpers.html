function to process records and store them to a variable or field. Callback must have one parameter for records. See example.

MLoadRecords

Loads records from a Fulcrum form

Description

Loads the form schema and creates dictionary to convert form_value keys to data_name keys. This will be modified soon to utilize the new query filtering options in LOADRECORDS.

Properties
Name Type Description
updateCallback function

Function to call to process retrieved records
recIDs Array

Array of string _record_ids to use in LOADRECORDS.
appFvs Object

Dictionary converting form_value keys to data_names
appRecords Array

Array of loaded records with form_value keys converted to data_name keys.
recordsOnLoad boolean

If true, records will be loaded immediately after form is loaded.
helpers FulcrumHelpers

Instance of FulcrumHelpers class to use helper methods. Defaults to M.
Examples
const appID = 'xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx'

// Callback function to process loaded records
function processRecordsCallback(){
 const loadedRecords = companionAppLoader.appRecords
 const titleList = loadedRecords.map(x=>x.scientific_name + ' - ' + x.common_name)
 SETVALUE('result',titleList.join('\n'))
}

const companionAppLoader = new MLoadRecords(null,appID,processRecordsCallback)

function changeSpecies(event){
  companionAppLoader.recIDs = $recordLinkField.map(x=>x.record_id) // update record ids from record link field
  companionAppLoader.processForm() // load records based on change event
}

ON('load-record',companionAppLoader.buildAppFvDict) // use if you want to load records later based on a change event
ON('change',$recordLinkField,companionAppLoader.loadRecords)
